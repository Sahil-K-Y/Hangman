# 🎮 Hangman Game

A classic CLI-based Hangman game built with Python. Test your vocabulary and try to guess the hidden word before the hangman is complete!

## 🚀 Features

- **Classic ASCII Art**: Visual representation of the hangman stages.
- **Large Word List**: Over 20+ tech-related words to guess.
- **Robust Logic**: Handles invalid inputs, repeated guesses, and case sensitivity.
- **Replayability**: Option to play multiple rounds without restarting the script.
- **Live Feedback**: Real-time display of used letters and remaining lives.

## 🛠️ Technologies Used

- **Python 3**: Core game logic and CLI interaction.

## 📖 How to Play

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sahil-K-Y/Hangman.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd Hangman
   ```
3. **Run the game**:
   ```bash
   python code.py
   ```
4. **Guess letters**: Enter one letter at a time to uncover the hidden word.
5. **Win or Lose**: You have 6 lives. If you guess the word, you win! If you run out of lives, the game is over.
